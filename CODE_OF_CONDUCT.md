# Code of Conduct

## Our Pledge

We aim to keep this project practical, respectful, and welcoming.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent when discussing bugs or routing mistakes.
- Keep examples safe to share publicly.
- Remove private paths, secrets, and private Skill contents before posting.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing another person's private information.
- Posting secrets, tokens, private repository data, or unauthorized Skill contents.
- Repeatedly pushing unrelated or out-of-scope features after maintainers clarify scope.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate these expectations.
